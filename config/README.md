# Device Configurations

This folder contains sanitized Cisco IOS configurations for the Enterprise OSPF Multi-Branch Network.

| File | Device |
|------|--------|
| `R1.txt` | Headquarters edge router |
| `R2.txt` | Branch 2 router |
| `R3.txt` | Branch 3 router |
| `S1.txt` | Headquarters access switch |
| `S2.txt` | Branch 2 access switch |
| `S3A.txt` | Branch 3 distribution switch |
| `S3B.txt` | Branch 3 access switch |
| `S3C.txt` | Branch 3 access switch |
| `ISP.txt` | Simulated ISP router |

> Password hashes, authentication keys, and device serial numbers were removed before publication.
